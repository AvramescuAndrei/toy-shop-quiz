# Toy Shop + Quiz (Node.js)

## Description
Node.js/Express web app using EJS templates and static assets, combining a toy shop (products + cart) with an interactive quiz.
Includes authentication, admin page, server-side validation, and SQLite-based persistence.

## Tech Stack
- Node.js + Express
- EJS + express-ejs-layouts
- SQLite (`cumparaturi.db`)
- JSON data (`intrebari.json`)

## How to Run
1. `npm install`
2. `node app.js`
3. Open `http://localhost:6789/`
